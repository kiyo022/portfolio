## 📚 プロジェクト一覧

### 🏢 顧客管理アプリ（Customer Management App）
**説明**: React × Supabase を用いた小規模 CRM（顧客管理）アプリ

**主な機能**:
- ✅ 顧客情報の管理（登録・編集・削除）
- ✅ 顧客検索・ソート機能
- ✅ メモ（対応履歴）管理
- ✅ レスポンシブ対応
- ✅ ダークモード対応

**使用技術**: React 18, TypeScript, Vite, Supabase, React Router

**ディレクトリ**: [`customer-management-app/`](./customer-management-app/)
